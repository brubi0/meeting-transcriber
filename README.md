## Requirements

- Modern web browser (Chrome, Edge, Firefox, Safari)

- Microphone

## Privacy

- Your API key is stored locally in your browser
- Transcripts are saved locally in browser storage
- Audio is sent to Deepgram for processing
- No data is sent to any other server

## Troubleshooting

**Microphone not working?**
- Check browser permissions (click the lock icon in address bar)
- Make sure microphone is not muted
- Try a different browser

**"Failed to start recording" error?**
- Check your API key is correct
- Make sure you have internet connection
- Try refreshing the page

**No transcription showing up?**
- Speak clearly and loudly
- Check that microphone is picking up audio
- Wait a few seconds for transcription to start

## Export Transcripts
